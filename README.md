## What's in Here

### Landing Pages
- **`summer_lock_in_page.html`** — Standalone HTML page for the Summer Lock In Challenge 2026. Paste directly into a GHL Custom HTML element or use as a full page body. Contains the GHL form embed (ID: `O3YVMeHBVbmcNwboiJgj`).

### Email Templates
- Located in `/email_templates/`. Each file is a standalone HTML email ready to paste into GHL workflows.

### Strategy & Reference Documents
- **Marketing Content Strategy Reference** — Master brand voice, messaging pillars, and content guidelines.
- **Adaptive Value Messaging Strategy** — Messaging frameworks tailored to different member segments.
- **Email Sequences** — Full copy for onboarding, re-engagement, and campaign sequences.
- **Member Testimonials** — Curated testimonials organized by theme for use in copy and ads.
- **Tech Management Reference** — GHL platform notes, workflow documentation, and technical SOPs.
- **GHL Onboarding Workflow** — Step-by-step documentation of the new member onboarding automation.

---

## Usage Notes

- All landing page files are written for **GoHighLevel (GHL)**. Paste the full HTML block into a Custom HTML element, or split CSS into Page Settings → Header Tracking Code.
- No phone number links (`sms:` hrefs) are used anywhere — all CTAs route to the opt-in form to maintain A2P compliance.
- SMS consent language and Privacy Policy / Terms of Service links are included on all opt-in pages.
